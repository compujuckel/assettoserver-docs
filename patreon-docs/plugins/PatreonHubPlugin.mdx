# PatreonHubPlugin
Connects this server to an AssettoServer Hub instance. Required for leaderboards of other plugins to work.

## Configuration
Enable the plugin in `extra_cfg.yml`
```yaml title="extra_cfg.yml"
EnablePlugins:
  - PatreonHubPlugin
```
