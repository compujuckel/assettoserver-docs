# RandomWeatherPlugin
Plugin that changes weather randomly.

:::note

For smooth weather transitions and rain you need to set `EnableWeatherFx` to `true` in `extra_cfg.yml`.

:::

## Configuration
Enable the plugin in `extra_cfg.yml`
```yaml title="extra_cfg.yml"
EnablePlugins:
- RandomWeatherPlugin
```
